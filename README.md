# Скрипт для поиска свежих статей на Хабре по ключевым словам и вывода их с датой и ссылкой.
 - (например: дизайн, фото, web, python).

Результат отображается в формате:
 - <дата> – <заголовок> – <ссылка>

Использует requests и BeautifulSoup. Подходит для мониторинга новых публикаций по интересующим темам.

# Дополнительное (необязательное) задание
 Улучшенная версия скрипта:
 - парсит список свежих статей (как раньше),
 - для каждой подходящей по preview статьи — загружает полную страницу,
 - ищет ключевые слова в полном тексте статьи,
 - выводит только те статьи, где ключевое слово найдено в самом контенте (а не только в превью).
